# Как запустить
Убедитесь, что находитесь в корневом директории проекта, и введите комаду в терминал, но рекомендую сперва сделать docker pool:
```shell
docker pull zhaisanbay/transaction-service
```
и потом
```shell
docker-compose up
```
ИЛИ
```shell
make build
```
